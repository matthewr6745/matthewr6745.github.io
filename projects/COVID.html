NYC COVID-19 Info

We were able to pull information from the NYC public health API, and display the number of new cases and death. It sounds quite simple but when sitting down to code it became increaslingly difficuly. One of the biggest issues we had was that after the user input the dates, we only wanted some of the information from the free API displayed, and not all. To work around that we coded so it limited what it would pull, to only the cases and death for the specific day.

If I were to add to this project, I would have some options. The first option would to be include a drop down bar so that the user can manually select a date instead of typing it in. Another option would be to include a calander so that the user could just select the date. Either way I go I would like to make the page look better, and not just a line of text on an empty white page.
